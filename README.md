# 🌱 Kisan Sahayak Portal

## 🚀 About the Project

**Kisan Sahayak Portal** is a modern web platform designed to help farmers access all essential services in one place.
It provides information about loans, government schemes, mandi prices, weather updates, and crop guidance — all through a clean and user-friendly interface.

👉 *"Sab kuch ek jagah — Digital Farming made simple."*

---

## 🎯 Key Features

* 🌾 **Fasal Guide** – Step-by-step crop growing guidance
* 💰 **Bank Loans** – Easy access to agricultural loans
* 🏛️ **Govt Schemes** – Latest schemes for farmers
* 📊 **Mandi Prices** – Real-time crop market rates
* 🌦️ **Weather Updates** – Location-based weather info
* 🤖 **AI Advisor** – Smart farming assistance
* 📰 **News Section** – Latest agriculture news
* 🏪 **Stores** – Farming products & resources
